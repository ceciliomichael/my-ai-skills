# My AI Skills

Welcome to my personal repository of AI skills! This repository contains custom skills that extend the capabilities of AI assistants.

## Included Skills

### Flat Design (`flat-design`)

This skill empowers the AI to create distinctive, production-grade flat UI interfaces that wow on first glance. 

**When to use:** Use this skill when asking the AI to build web components, pages, artifacts, posters, or applications (websites, landing pages, dashboards, React components, HTML/CSS layouts, or any web UI).

**Why this is different:**
Most AI design tools and instructions lean heavily on "safe" trends like gradients, subtle drop shadows, and glassmorphism, often resulting in generic, uninspired outputs. This skill explicitly rejects those tropes. It forces the AI to rely purely on **bold typography, strategic color blocking, strong contrast, layout rhythm, and motion**. The result isn't just a "clean" UI—it's a striking, structurally sound, and highly opinionated flat design that feels premium, energetic, and intentionally crafted.

**Features:**
- Outputs high-impact, emotionally resonant flat UI.
- Focuses purely on typography, color blocking, contrast, and rhythm.
- Uses light mode by default.
- Strictly no gradients, soft drop shadows, or glassmorphism.
